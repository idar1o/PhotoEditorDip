# 📸 PhotoEditorDip

**PhotoEditorDip** — это мобильное приложение для редактирования изображений, построенное с использованием **Clean Architecture** и библиотеки **OpenCV**. Приложение предоставляет мощные инструменты для обработки изображений, такие как фильтры, коррекция цвета, размытие, резкость, и многое другое.

---

## 🛠 Технологии и архитектура

* **Clean Architecture**
  Проект разделён на три уровня:

  * **Presentation** — UI, управление состоянием (Jetpack Compose + ViewModel).
  * **Domain** — бизнес-логика, use-cases.
  * **Data** — работа с файлами, изображениями, доступ к API OpenCV.

* **OpenCV (Open Source Computer Vision Library)**
  Используется для обработки изображений, таких как:

  * Гамма-коррекция
  * Размытие (Gaussian Blur, Median Blur)
  * Повышение резкости (Unsharp Masking)
  * Наложение эффектов (Sepia, Vignette, Solarize и др.)
  * Обработка контрастности, яркости и насыщенности

---

## 🔍 Основные возможности

* 📷 Загрузка изображений из галереи
* ✂️ Обрезка и изменение размера
* 🎨 Применение фильтров и эффектов (с предварительным просмотром)
* 📁 История действий (Undo / Redo)
* 💾 Сохранение обработанных изображений
* 💡 Система пресетов: пользователь может сохранить свой набор эффектов

## 📂 Структура проекта (Clean Architecture)

```
PhotoEditorDip/
├── presentation/         # UI: Compose, ViewModels, State
├── domain/               # UseCases, Entities, Interfaces
├── data/                 # Реализация интерфейсов, OpenCV обработка
├── di/                   # Hilt-модули для зависимостей
└── utils/                # Вспомогательные классы и константы
```

## 🚀 Как запустить

1. Склонируйте репозиторий:

```bash
git clone https://github.com/idar1o/PhotoEditorDip.git
```

2. Откройте проект в **Android Studio Arctic Fox+**
3. Убедитесь, что у вас установлен **NDK** и настроен **OpenCV Android SDK**
4. Запустите на эмуляторе или физическом устройстве

---

## 🔧 Требования

* Android 7.0+
* OpenCV 4.x
* Kotlin 1.8+
* Jetpack Compose
* Hilt (для внедрения зависимостей)
