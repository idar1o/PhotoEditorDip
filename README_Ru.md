📸 PhotoEditorDip
PhotoEditorDip — это Android-приложение, разработанное с использованием Kotlin, поддержкой многомодульной архитектуры, Clean Architecture, и лучших практик Android-разработки. Цель проекта — предоставить удобный и расширяемый редактор фотографий.

🚀 Особенности
🧱 Многомодульная структура — для масштабируемости и удобства поддержки

✅ Чистая архитектура (Clean Architecture) — разделение ответственности и слоёв: Domain, Data, Presentation

💡 MVVM-паттерн — для реактивного управления состоянием

🗂 DI (Hilt) — внедрение зависимостей по стандартам Android Jetpack

🖼 Jetpack Compose — современный UI-фреймворк от Google (если используется)

📷 Инструменты для базового редактирования фото (обрезка, фильтры, поворот и т.д.)

UI: Jetpack Compose / XML

База данных: Room (если используется)

Навигация: Jetpack Navigation
